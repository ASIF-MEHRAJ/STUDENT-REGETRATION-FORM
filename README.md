Student Registration Form
A clean, semantic HTML-based registration form designed to capture student details efficiently. This project focuses on utilizing proper HTML form structures, including grouping related inputs and ensuring accessibility through labels.

🚀 Features
Categorized Data Entry: Uses <fieldset> and <legend> tags to logically group Personal, Address, and Course details.

User-Friendly Inputs: Includes placeholder text to guide users on what information to enter.

Selection Menus: Implements <select> dropdowns for gender, city, course tracks, and duration to ensure data consistency.

Document Upload: Includes a file input for students to submit necessary documentation.

Accessible Design: Uses <label> tags with for attributes correctly linked to input ids for better screen reader support.

📂 Form Sections
Personal Details: Captures Name, Contact, Age, and Gender.

Address Details: Includes Permanent Address, Temporary Address, and City selection.

Course Details: Allows users to select their desired course track and duration (1–5 years).

Submission: Includes a document upload field and a formal submit button.

🛠️ Built With
HTML5: The backbone of the structure, focusing on semantic elements like main, form, and fieldset.

📝 How to Use
Clone the repository or download thIS html file.

Open the file in any modern web browser.

Fill out the required fields and interact with the dropdown menus.
